RESIDE 실행 순서 (권장)
=======================

1) 환경 변수 확인
- .env 에 API_FOOTBALL_KEY, U25_API_TOKEN, U25_CORS_ORIGINS 설정

2) 설정 확인
- data/automation_config.json 확인
  - schedule_times = ["10:00"]
  - timezone = "Asia/Seoul"

3) 기본 운영 시작 (권장)
- RESIDE_전체자동.bat 실행
- API + 스케줄러 + 라이브피드가 함께 시작됨

4) 로그온 자동 시작이 필요하면
- RUN_U25_AT_LOGON.cmd 를 작업 스케줄러 "로그온 시"로 등록

5) 단발 10시 작업을 별도로 쓰려면 (선택)
- register_daily_10am_task.bat 로 작업 등록
- enable_daily_10am_task.bat 로 활성화
- disable_daily_10am_task.bat 로 비활성화

주의
- Full 상시 운영과 10시 단발 작업을 동시에 켜면 중복 실행 가능
- Full 운영이 메인이라면 10시 단발 작업은 비활성화 권장
