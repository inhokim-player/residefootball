RESIDE 점검 체크리스트
======================

[매일 확인]
- API 열림 확인: http://127.0.0.1:8010/health
- 스케줄러 동작 확인: u25_auto_scheduler 창/로그
- live_feed 동작 확인: live_feed 로그 갱신
- site/index.html 에 데이터 반영 확인

[설정 확인]
- data/automation_config.json 의 schedule_times/timezone 값
- .env 의 API_FOOTBALL_KEY 유효성
- 공개 서버라면 allow_local_token_bypass=false 권장

[작업 스케줄러 확인]
- 단발 작업명: U25_RESIDE_DailyPipeline_10am
- Full 운영 중에는 disable_daily_10am_task.bat 상태 유지 권장

[문제 시 빠른 복구]
- Full 재시작: RESIDE_전체자동.bat
- 단발 파이프라인 수동 1회: run_u25_pipeline_10am_once.bat
- 수동 업데이트: run_u25_update_once.bat
