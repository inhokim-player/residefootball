RESIDE 스케줄 모드 전환
========================

[모드 A: Full 상시 운영 (권장)]
- 실행: RESIDE_전체자동.bat
- 특징: API + 스케줄러 + 라이브피드 일체 운영
- 중복방지: 실행 시 U25_RESIDE_DailyPipeline_10am 자동 비활성화

[모드 B: 윈도우 10시 단발 작업]
- 등록: register_daily_10am_task.bat
- 활성화: enable_daily_10am_task.bat
- 비활성화: disable_daily_10am_task.bat
- 특징: 매일 로컬 시계 10:00에 u25_pipeline --once 1회

[전환 규칙]
- A -> B: Full 운영을 중지하고 enable_daily_10am_task.bat 실행
- B -> A: disable_daily_10am_task.bat 실행 후 RESIDE_전체자동.bat 실행

[권장]
- 실제 운영은 모드 A 하나로 고정
- 모드 B는 예비/백업용으로만 사용
