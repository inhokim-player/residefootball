## 주요 엔드포인트(백엔드)

| 메서드 | 경로 | 비고 |
| --- | --- | --- |
| GET | `/health` | `U25_API_TOKEN` 설정 여부만 `auth_required`로 알림 |
| GET | `/system/admin-signals` | 운영용 파일 mtime·크기(내용 없음) |
| GET | `/system/automation-status` | 스케줄·타임존·스케줄러 state |
| GET | `/system/status` | DB 요약 |

`U25_API_TOKEN`이 설정된 경우:

- 기본적으로 `Authorization: Bearer …` 필요(로컬호스트 우회 정책은 `automation_config.json` / 환경변수 참고).

## 환경 변수(이름만 — 값은 .env)

| 변수 | 용도 |
| --- | --- |
| `API_FOOTBALL_KEY` | API-Football 호출 |
| `U25_API_TOKEN` | 공개 API 잠금(Bearer) |
| `U25_CORS_ORIGINS` | 배포 시 CORS 화이트리스트 |
| `U25_EXPOSE_DOCS`, `U25_ALLOW_LOCAL_TOKEN_BYPASS` 등 | `.env.example` 참고 |

## 로컬 실행 예시

```bash
cd /path/to/project
python -m pip install -r requirements.txt
python -m uvicorn u25_api:app --host 127.0.0.1 --port 8010
```

`site/index.html` 의 기본 API 베이스는 `http://127.0.0.1:8010` 이며, `data/automation_config.json` 의 `api.host` / `api.port` 와 맞춥니다.
